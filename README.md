## Tech Stack

- React 19.2.3
- TypeScript 5.9.3
- Vite 7.2.4
- Tailwind CSS 4.1.17
- Lucide React (icons)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Deploy to GitHub Pages

```bash
npm run deploy
```

The application will be built and deployed to GitHub Pages automatically.
